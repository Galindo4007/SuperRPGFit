Para poder abrir correctamente el archivo completamente funcional, debera de tener dos ventanas de Visual Studio Code (puede abrir Visual Studio Code como normalmente lo abre, una vez que se abra la primera pagina, teclee en su teclado la combinacion ctrl + shift + n), en una de las dos ventanas debera de abrir este folder y en la otra ventana el folder de la API (shortcut: ctrl + k + o).
Dentro de la ventana de la API, debera de abrir la terminal (ctrl + Ã±) y debera de escribir "npm start" le deberia de aparecer un texto en verde y un mensaje parecido a "la api esta corriendo en el puerto 5500". Todo listo por la parte de la api.
Dentro de este folder, abra el archivo index.html que esta dentro de la carpeta html. Despues, debera de tener la extension de "live server" para que se pueda abrir como servidor y no como un solo archivo (ctrl + shift + x, para abrir la ventana de extensiones y poder descargarlo). Una vez descargado, abajo a la derecha, donde la barrita azul de Visual Studio Code, aparece el boton "Go Live" junto con un icono, hace click en ese boton y se abrira el archivo en su navegador de preferencia, desde ahi ya podra ver correctamente la pagina web.
Si desea ver completamente funcional la pagina, debera de abrir la carpeta de la base de datos dentro de la aplicacion de sql management studio, abre el archivo "basedatos" lo ejecuta, despues el archivo "datosprueba" y ejecuta los que desee utilizar sino es que completo, abre el archivo "procedures" y ejecuta procedure por procedure para que sirva correctamente y con esto ya deberia de estar correctamente funcional toda la pagina web.
